## 🧩 Project Showcase

<details open>
<summary><strong>🔬 RV32I RISC-V CPU Processor &nbsp;&nbsp</strong></summary>
<br/>

> Full silicon-to-FPGA implementation of a 32-bit RISC-V processor: designed, verified, and synthesised solo.

| | |
|:---|:---|
| **Architecture** | 8 modular RTL components: 32×32 register file, combinational decoder, ALU, configurable shifter, 8-state multicycle FSM |
| **Instruction Set** | All 40 RV32I base instructions implemented and independently verified |
| **Verification** | UVM environment with constrained-random stimulus, self-checking scoreboards → **100% functional coverage, 100% pass rate** |
| **Synthesis** | DE10-Lite FPGA @ 50 MHz · **< 3% logic utilisation** · 16 KB on-chip data memory |
| **Stack** | Verilog · SystemVerilog · UVM · Quartus Prime · ModelSim |

<details>
<summary><strong>🧠 GNN-based Static Timing Analysis &nbsp;|&nbsp; <em>AI for EDA</em></strong></summary>
<br/>

> Applying Graph Neural Networks to predict timing violations in digital circuits — accelerating EDA without full sign-off.

| | |
|:---|:---|
| **Problem** | Sign-off static timing analysis is computationally expensive at scale |
| **Approach** | Model the netlist as a graph; train GNNs to predict timing hotspots at RTL stage |
| **Impact** | Enables early-stage violation detection, reducing downstream EDA iteration cycles |
| **Stack** | Python · PyTorch · PyTorch Geometric · NetworkX |
